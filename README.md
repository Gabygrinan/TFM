# TFM

Repositorio para el Trabajo de Fin de Máster (TFM).

## Estructura del proyecto

```
TFM/
├── data/           # Datos del proyecto
│   ├── raw/        # Datos originales sin procesar
│   └── processed/  # Datos procesados
├── notebooks/      # Jupyter Notebooks de análisis y experimentación
├── src/            # Código fuente del proyecto
├── requirements.txt
└── README.md
```

## Configuración del entorno local

### Requisitos previos

- Python 3.9 o superior
- Git

### Instalación

1. Clona el repositorio:

   ```bash
   git clone https://github.com/Gabygrinan/TFM.git
   cd TFM
   ```

2. Crea y activa un entorno virtual:

   ```bash
   python -m venv venv
   # En Windows:
   venv\Scripts\activate
   # En Linux/macOS:
   source venv/bin/activate
   ```

3. Instala las dependencias:

   ```bash
   pip install -r requirements.txt
   ```

## Uso

Consulta los notebooks en la carpeta `notebooks/` para ver los análisis realizados.
